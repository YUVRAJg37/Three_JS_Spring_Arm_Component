## Run Locally

Clone the project

```bash
  git clone https://github.com/YUVRAJg37/Three_JS_Spring_Arm_Component
```

Go to the project directory

```bash
  cd Three_JS_Spring_Arm_Component
```

Install dependencies

```bash
  npm install
```

Start the application

```bash
  npm run dev
```
